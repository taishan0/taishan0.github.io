ras是一个非常神秘的算法，那么它神秘在哪里 请少侠自己摸索！ 注意：得到的 flag 请包上 flag{} 提交

把pub.key的内容放进公钥解析中解析一下
SSL在线工具-在线公钥解析-在线公钥提取-公钥在线破解-SSLeye官网

RSA求解明文，安利函数pow()_rsa中pow函数-CSDN博客（这里感谢CTFer菜菜给的教程）

这时候就可以编写脚本了

pip3 install rsa
