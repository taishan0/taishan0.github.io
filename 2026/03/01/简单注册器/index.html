简单注册器

生活中难免会有需要使用一些付费的程序，但是没有绿色版怎么办？只能自己逆向看看注册程序的代码是什么逻辑了。 注意：得到的 flag 请包上 flag{} 提交

直接下载解压后发现是个APK文件

可反编译查看源码，使用工具JADX、AndroidKiller、Android studio等工具，我这里使用的是JADX

打开后依旧直接找到main函数的代码页（我这里没截完）

分析这串代码我们可以知道

这里是最重要的

对固定字符数组 x = "dd2940c04462b4dd7c450528835cca15".toCharArray() 执行 4 次逐位计算，关键是按 ASCII 码值运算（字符本质是 ASCII 数字）：

  • 计算 1：x[2] = (x[2] + x[3]) - 50 → 把索引 2 的字符换成运算后结果；
  • 计算 2：x[4] = (x[2] + x[5]) - 48 → 基于计算 1 的结果更新索引 4；
  • 计算 3：x[30] = (x[31] + x[9]) - 48 → 用索引 31、9 的值更新索引 30；
  • 计算 4：x[14] = (x[27] + x[28]) - 97 → 用索引 27、28 的值更新索引 14；
  • 所有计算都是 “字符 ASCII 值相加后减固定数”，再转回字符。
  • 对计算后的数组执行对称反转（32 位数组，前 16 位和后 16 位交换）：
    • 循环 i从0到15，交换 x[i] 和 x[31-i] → 本质是把整个数组倒序；
    • 反转后拼接成字符串，包裹flag{}后就是最终结果。

本人java是小白，所以这里用的python（虽然python也没学得多明白）

运行代码后即可得到flag

flag{59acc538825054c7de4b26440c0999dd}

其实这题不用费脑子猜输入，就盯着程序里那串固定字符：先改 4 个位置的字符（按数字加减算），再把整串倒过来，套个flag{}就是最后的答案。
